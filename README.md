# correlation_analysis-
investment correlation analysis excel visualizations.
## Objective :

The objective of this task is to identify the strength and direction of relationships between key variables such as age, investment duration, and expected returns.

## Tools Used :

Microsoft Excel

Data Analysis ToolPak

Scatter Plot

Bubble Chart

## Methodology :

The correlation analysis was performed by selecting numeric variables including Age, Duration (converted to numeric values), and Expectation (converted to numeric values). Using Excel’s Data Analysis ToolPak, a correlation matrix was generated to measure the relationship between these variables.

Scatter plots were used to visually examine the relationship between Age and Duration as well as Age and Investment Expectations. A bubble chart was also created to represent the relationship between investment duration and expectations, with bubble size representing participant age.

## result :

The correlation matrix provides insights into how strongly the variables are related. Positive correlation values indicate that the variables move in the same direction, while negative values indicate an inverse relationship.

## Conclusion :

The correlation analysis helps in understanding how demographic factors and investment duration influence investment expectations. This analysis demonstrates the application of Excel data analysis tools to identify patterns and relationships within financial datasets.
